# claude

## CLI Reference

```
claude        # Start Claude Code
cd folder     # Move into a folder
cd ..         # Move back
ls            # List files (Mac)
dir           # List files (Windows)
pwd           # Show current path (Mac)
```

## Example: Start a New Project

```
mkdir my-portfolio   # Create a new project folder
cd my-portfolio      # Move into it
claude               # Start Claude Code inside the project
```
